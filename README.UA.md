# Корисник (Korysnyk)

Мобільний та веб-застосунок для сканування штрих-кодів продуктів з аналізом складу та оцінкою безпеки для здоров'я.

*Назва походить від українського "корисний" + суфікс "-ник" — помічник у користі для свідомого вибору продуктів.*

## Опис проєкту

Платформа надає споживачам інструменти для прийняття інформованих рішень про купівлю продуктів харчування. Користувачі сканують штрих-код і отримують детальний аналіз інгредієнтів, включаючи потенційні ризики для здоров'я та персоналізовані рекомендації.

## Функціональність

**Мобільний застосунок**
- Сканування штрих-кодів у режимі реального часу
- Відображення складу продукту з розшифруванням інгредієнтів
- Оцінка безпеки та впливу на здоров'я
- Визначення алергенів та попередження
- Інформація про харчову цінність
- Історія сканувань та обране
- Базова функціональність в офлайн-режимі

**Веб-платформа**
- База даних продуктів з розширеним пошуком
- Статті та дослідження щодо безпеки продуктів
- Спільнота користувачів та відгуки
- Інструменти для порівняння продуктів
- Фільтрація за критеріями здоров'я
- Користувацький контент та рейтинги
- Бібліотека матеріалів про здоров'я та харчування

**Технологічні можливості**
- Кросплатформенна розробка (Flutter)
- Адаптивний веб-інтерфейс
- База даних продуктів та інгредієнтів
- Алгоритми класифікації безпеки на основі машинного навчання
- Аутентифікація та управління профілями користувачів
- Модерація контенту
- Інтеграція з базами даних безпеки продуктів харчування
- Синхронізація даних у реальному часі

## Цільова аудиторія

Платформа розроблена для споживачів, які прагнуть до усвідомленого вибору продуктів харчування:
- Люди з дієтичними обмеженнями та алергіями
- Батьки, які обирають безпечні продукти для сім'ї
- Споживачі, що контролюють хронічні захворювання (діабет, підвищений холестерин)
- Спільнота здорового способу життя

## Технічний стек

**Frontend**
- Flutter для мобільних застосунків (iOS/Android)
- React/Next.js для веб-платформи
- Responsive design для всіх пристроїв

**Backend**
- Python/FastAPI
- PostgreSQL для зберігання даних
- Redis для кешування

**ML/AI**
- Інтеграція LLM для аналізу інгредієнтів
- Алгоритми оцінки безпеки
- Персоналізація рекомендацій

**Інфраструктура**
- AWS/Google Cloud Platform
- Docker для контейнеризації
- CI/CD pipeline

## База даних

Платформа містить інформацію про 1250+ інгредієнтів з характеристиками:
- Класифікація за категоріями
- Потенційний вплив на здоров'я
- Алергенні властивості
- Рекомендації для різних груп споживачів
- Інтеграція з OpenFoodFacts та локальними базами даних

## Персоналізація

- Хронічні захворювання
- Дієтичні переваги
- Історія реакцій на продукти
