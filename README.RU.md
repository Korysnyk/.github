# Корисник (Korysnyk)

Мобильное и веб-приложение для сканирования штрих-кодов продуктов с анализом состава и оценкой безопасности для здоровья.

*Название происходит от украинского "корисний" (полезный) + суффикс "-ник" — помощник в пользе для осознанного выбора продуктов.*

## Описание проекта

Платформа предоставляет потребителям инструменты для принятия информированных решений о покупке продуктов питания. Пользователи сканируют штрих-код и получают детальный анализ ингредиентов, включая потенциальные риски для здоровья и персонализированные рекомендации.

## Функциональность

**Мобильное приложение**
- Сканирование штрих-кодов в режиме реального времени
- Отображение состава продукта с расшифровкой ингредиентов
- Оценка безопасности и влияния на здоровье
- Определение аллергенов и предупреждения
- Информация о пищевой ценности
- История сканирований и избранное
- Базовая функциональность в офлайн-режиме

**Веб-платформа**
- База данных продуктов с расширенным поиском
- Статьи и исследования по безопасности продуктов
- Сообщество пользователей и отзывы
- Инструменты для сравнения продуктов
- Фильтрация по критериям здоровья
- Пользовательский контент и рейтинги
- Библиотека материалов о здоровье и питании

**Технологические возможности**
- Кросс-платформенная разработка (Flutter)
- Адаптивный веб-интерфейс
- База данных продуктов и ингредиентов
- Алгоритмы классификации безопасности на основе машинного обучения
- Аутентификация и управление профилями пользователей
- Модерация контента
- Интеграция с базами данных безопасности продуктов питания
- Синхронизация данных в реальном времени

## Целевая аудитория

Платформа разработана для потребителей, которые стремятся к осознанному выбору продуктов питания:
- Люди с диетическими ограничениями и аллергиями
- Родители, выбирающие безопасные продукты для семьи
- Потребители, контролирующие хронические заболевания (диабет, повышенный холестерин)
- Сообщество здорового образа жизни

## Технический стек

**Frontend**
- Flutter для мобильных приложений (iOS/Android)
- React/Next.js для веб-платформы
- Responsive design для всех устройств

**Backend**
- Python/FastAPI
- PostgreSQL для хранения данных
- Redis для кэширования

**ML/AI**
- Интеграция LLM для анализа ингредиентов
- Алгоритмы оценки безопасности
- Персонализация рекомендаций

**Инфраструктура**
- AWS/Google Cloud Platform
- Docker для контейнеризации
- CI/CD pipeline

## База данных

Платформа содержит информацию о 1250+ ингредиентах с характеристиками:
- Классификация по категориям
- Потенциальное влияние на здоровье
- Аллергенные свойства
- Рекомендации для различных групп потребителей
- Интеграция с OpenFoodFacts и локальными базами данных

## Персонализация

- Наличие аллергий
